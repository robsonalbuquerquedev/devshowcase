Passo a passo do código

por Robson Albuquerque

Atualizado em 19/Janeiro/2023

Função "criar_pastas():"

É chamada no início do código e os nomes das pastas são passados como argumentos.

Ela usa o módulo "os" para verificar se as pastas já existem, e se não, cria-as. Isso garante que as pastas necessárias para armazenar as imagens estejam disponíveis antes de baixar e salvar as imagens.

Em seguida, as URLs das imagens de gatos e cachorros são definidas em listas separadas.

Função "baixar_imagens()":

É chamada e os argumentos necessários são passados para esta função.

Ela usa a biblioteca "requests" para baixar as imagens a partir das URLs fornecidas e salva-as na pasta especificada.

Dessa forma, as imagens de gato e cachorro são baixadas e salvadas nas pastas correspondentes, "pasta1" para gatos e "pasta2" para cachorros.

Função "esconder_imagens()":

É chamada para combinar as imagens de gato e cachorro, lado a lado, e salvar a imagem resultante na pasta3.

Ela usa a biblioteca PIL para abrir as imagens de gato e cachorro, redimensiona a imagem de cachorro para o tamanho da imagem de gato, cola a imagem de cachorro sobre a imagem de gato e salva a imagem resultante na pasta3.

Isso permite esconder a imagem de cachorro dentro da imagem de gato.

Função "extrair_imagens()":

É chamada para extrair as imagens escondidas da pasta3.

Ela usa a biblioteca PIL para abrir cada imagem resultante e separa as bandas de cor (R, G, B) e salva cada banda em uma pasta diferente (pasta_cat, pasta_dog).

Desta forma, as camadas de imagem escondidas são extraídas e salvas em pastas separadas.

Observações

É importante lembrar que é necessário fazer o upload das imagens para o seu notebook Colab antes de executar o código e você também deve criar as pastas na sua unidade de armazenamento do Colab (geralmente é a unidade "drive/") antes de rodar o código.
