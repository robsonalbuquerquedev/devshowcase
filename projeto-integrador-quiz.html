get_resposta_usuario(pergunta)

Serve para o usuário inserir a resposta e valida se ela é válida.

Outras Informações

get_resposta_usuario(pergunta)

é chamada no momento em que é necessário a resposta do usuário, ela espera como parametro pergunta para que assim possa verificar as alternativas possíveis para a resposta. Ela valida se a resposta é válida, pedindo novamente caso não esteja entre as alternativas.

exibir_pergunta(pergunta)

é chamada para exibir a pergunta e as alternativas para o usuário. Ela espera como parametro pergunta para que assim possa pegar as informações e exibi-las na tela.

exibir_resultado(score, total_perguntas)

é chamada no final do programa, ela é utilizada para calcular o percentual de questões corretas e exibir a pontuação final do usuário. Ela recebe como parametros o score e o total de perguntas para poder calcular o percentual.

run_quiz()

é a função principal e é chamada para iniciar a execução do programa. Ela contém a lógica do programa, inicializa as variáveis score e total_perguntas, contém as perguntas e alternativas do quiz, e o loop para percorrer as perguntas.

Dentro do loop é chamada as funções exibir_pergunta, get_resposta_usuario e verifica a resposta se está correta, incrementando o score quando necessário. No final do loop é chamada a função exibir_resultado para exibir o resultado.
